# Automac Free

**Your free AI PC doctor — right inside VS Code.**

Click a problem. Get a diagnosis. That's it.

## What It Does

Automac Free scans your Windows PC with 13 read-only tools and tells you what's wrong in plain English. It never changes anything on your system.

**Pick a problem card:**
- My computer is slow
- Running out of disk space
- Something keeps crashing
- Slow to boot
- Windows Update issues
- Network problems
- What is this process?
- Something is wrong but I don't know what

Or just type your own question.

## Privacy

- Everything stays between you and your Copilot subscription
- Personal data (IPs, emails, usernames) is scrubbed automatically
- No telemetry, no tracking, no data collection

## Requirements

- Windows 10 or 11
- VS Code 1.85+
- GitHub Copilot

## Want More?

Automac Free is read-only by design. For automated fixes and advanced diagnostics, check out [Autodeck](https://marketplace.visualstudio.com/items?itemName=beautifulplanet.autodeck-basic-it-free).
