# Automac Free

**Your free AI PC doctor — right inside VS Code.**

Click a problem. Get a diagnosis. That's it.

## What It Does

Automac Free scans your Windows PC with 13 diagnostic tools and tells you what's wrong in plain English. It never changes anything on your system — it just looks and reports back.

**Pick a problem card:**
- My computer is slow
- Running out of disk space
- Something keeps crashing
- Slow to boot
- Windows Update issues
- Network problems
- What is this process?
- Something is wrong but I don't know what

Or just type your own question.

## Privacy

- Everything stays between you and your Copilot subscription
- Personal data (IPs, emails, usernames) is scrubbed automatically
- No telemetry, no tracking, no data collection

## Don't Trust Us — Audit Us

This extension is open source. You can verify exactly what it does before you install it.

**Audit with GitHub Copilot (takes 2 minutes):**

1. Open this repo in GitHub and start a Copilot Chat
2. Ask: *"Does this extension collect any telemetry, phone home, or send data anywhere besides the user's own Copilot subscription?"*
3. Ask: *"What PowerShell commands does it run? Are any of them destructive?"*
4. Ask: *"Show me the PII scrubbing code — what does it strip before sending to the AI?"*

Copilot can read every file in this repo. There are no hidden endpoints, no analytics, no tracking pixels. The code is the proof.

**What you'll find:**
- 13 read-only diagnostic commands — none of them change your system
- PII scrubbing strips IPs, emails, MAC addresses, API keys, JWTs, and connection strings before anything leaves your machine
- The only network call is to your own GitHub Copilot subscription
- No telemetry SDK, no fetch/axios calls to external servers, no data collection of any kind

## Requirements

- Windows 10 or 11
- VS Code 1.85+
- GitHub Copilot
