# Changelog

## [0.3.0] - 2026-04-18

### Added
- Sidebar icon — Automac Free now has its own icon in the VS Code activity bar
- Clicking the sidebar icon opens the diagnostics panel directly
- CLIXML error parsing — PowerShell errors now show clean messages instead of raw XML

## [0.2.2] - 2026-04-17

### Changed
- Updated README with Automac IT comparison table and upgrade info
- Updated marketplace description

## [0.2.0] - 2026-04-16

### Added
- Custom dark theme (app-like, not raw VS Code)
- Thinking animation while AI works
- Status bar indicator
- Welcome screen on first launch
- Dev mode setting to switch back to VS Code theme
- Card descriptions on home screen

## [0.1.0] - 2026-04-16

### Added
- Initial release
- 13 read-only diagnostic tools (system info, processes, startup items, temp files, disk usage, disk health, event logs, installed software, network config, network test, Windows Update, large files, battery health)
- 8 one-click diagnostic cards
- Custom question input
- AI-powered analysis via GitHub Copilot
- PII scrubbing for privacy (IPs, emails, MAC addresses, API keys, JWTs, connection strings)
- CSP nonce security for webview
- Input sanitization against injection
- Markdown-formatted AI responses
